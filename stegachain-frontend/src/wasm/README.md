# src/wasm — WASM Engine Artifacts

This folder contains the compiled output of the `stegachain-engine` Rust crate,
produced by `wasm-pack build --target web`.

## Files

| File | Description |
|---|---|
| `stegachain_engine.js` | JS glue code generated by wasm-bindgen |
| `stegachain_engine.d.ts` | TypeScript declarations for the JS glue |
| `stegachain_engine_bg.wasm` | Compiled WebAssembly binary |
| `stegachain_engine_bg.wasm.d.ts` | TypeScript declarations for the raw WASM exports |

## Updating

When the Rust engine changes, rebuild and copy the updated artifacts here:

```bash
# From the monorepo root
cd stegachain-engine
wasm-pack build --target web --release

# Then copy into the frontend
cp pkg/stegachain_engine.js     ../stegachain-frontend/src/wasm/
cp pkg/stegachain_engine.d.ts   ../stegachain-frontend/src/wasm/
cp pkg/stegachain_engine_bg.wasm ../stegachain-frontend/src/wasm/
cp pkg/stegachain_engine_bg.wasm.d.ts ../stegachain-frontend/src/wasm/
```

## Do not edit these files manually.
